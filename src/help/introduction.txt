                      WELCOME TO APOPHYSIS-J

Apophysis-j is a port of the fractal flame editor Apophysis to java.
As such, it should run on any machine equipped with a java runtime
environment (jre) 1.5 or higher.




Official site :  https://github.com/mebigfatguy/apophysis-j/wiki

Mailing list  :  http://tech.groups.yahoo.com/group/Apophysis-j/

Resources     :  http://apophysis-j.deviantart.com
